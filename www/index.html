Painless, JavaScript-free AJAX

Ajaxial is a single-file web library that embeds AJAX instructions directly in your HTML structure, allowing you to painlessly design client-server interactions without ever touching JavaScript yourself.

With AJAX at your fingertips, the web becomes more powerful than all of Greece!

Powerfully Extensible

By defining nearly all of its key functionality modularly, Ajaxial opens itself up to extreme customizability for any situation you might need, from retrieving <template> elements instead of making HTTP requests, to custom processing of response content, and more.

Deeply Event-Driven

Everything in Ajaxial runs on events from top to bottom, allowing seamless integration with any other libraries or frameworks that expose their own events, or can consume Ajaxial's events.

This example uses two other libraries that pair excellently with Ajaxial, Facet for web components and highlight.js for syntax highlighting.
